# 启航电商即时配送系统
## 一、项目介绍

启航电商即时配送系统-商城+跑腿构建自己的即时零售体系。

主体功能店铺管理、订单管理、订单履约、跑腿配送、售后管理等功能。系统连锁店铺范围配送。

本项目采用Java SpringBoot开发，新电商时间为您构建即时零售体系，有什么需求和建议请联系作者。


## 二、主要使用到的技术
+ Java17
+ SpringBoot3
+ Redis（缓存、消息队列）
+ MyBatis-Plus
+ MySQL
+ Vue
+ ElementUI
